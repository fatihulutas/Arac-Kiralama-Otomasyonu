# Araç Kiralama Otomasyonu

Java programlama dilinin temel yapı taşları (Diziler, Döngüler ve Algoritmalar) kullanılarak geliştirilmiş, konsol tabanlı bir araç kiralama yönetim sistemidir.

Bu proje; veritabanı veya karmaşık kütüphaneler kullanmadan, verilerin **Matris (2D Array)** yapısında nasıl yönetileceğini ve bir işletme mantığının (indirim, stok takibi, ciro hesaplama) kod tarafında nasıl kurgulanacağını simüle eder.

### Temel Özellikler
* **Araç Yönetimi:** Ekleme, Silme ve Listeleme.
* **Filtreleme:** Müsaitlik durumu ve Yakıt türüne (Benzin/Dizel/Elektrik) göre arama.
* **Kiralama Mantığı:** Gün bazlı fiyat hesaplama ve **uzun dönem (7+ gün) için otomatik %15 indirim**.
* **Raporlama:** Yapılan kiralamaların geçmişini ve toplam ciro takibini görüntüleme.
